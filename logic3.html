The Missing Peso Puzzle

Three friends check into a hotel room costing Php 3,000.

• They each contribute Php 1,000 (Php 1,000 × 3 = Php 3,000).

• Later, the hotel realizes the room was only Php 2,500, so the manager gives Php 500 to the bellboy to return to them.

• The bellboy, however, keeps Php 200 for himself and gives each guest Php 100 back.

Now each guest has paid Php 900 (Php 1000 - 100 = Php 900), totaling Php 2,700.

The bellboy kept Php 200, bringing the total to Php 2,900.

Where is the missing Php 100?
